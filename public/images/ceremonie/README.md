# Assets — Images

Ce dossier est réservé aux images locales du projet.

Pour l'instant, les images sont servies via **picsum.photos** (placeholders thématiques).

En production, remplacer les URLs picsum dans `src/constants/index.ts` par les chemins
vers les images locales optimisées (WebP, dimensions adaptées).

## Convention de nommage

- `hero-bg.webp` — Image de fond du hero
- `service-anniversaire.webp` — Image de la carte service
- `gallery-01.webp` — Images de la galerie
- Toujours en minuscules, tirets pour les espaces
